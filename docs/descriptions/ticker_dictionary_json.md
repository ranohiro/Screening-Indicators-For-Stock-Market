# テーマ・タグマスター辞書（`ticker_dictionary.json`）構造・仕様書

このドキュメントは、日本株投資アプリのトレンドテーマ分析機能において、各銘柄とテーマタグ（セクターや技術キーワード）との紐付けを定義する JSON マスタファイル [ticker_dictionary.json](file:///Users/hiranotakahiro/Projects/銘柄スクリーニング検証/data/ticker_dictionary.json) のデータ構造と仕様について説明したものです。

---

## 1. 概要
`ticker_dictionary.json` は、全上場銘柄（約4,000社）に対して、その企業がどのようなビジネスセクター、エンドマーケット、コア技術、またはイベント/地政学に関連しているかを、多様な「テーマタグ」として体系化した辞書データです。

このファイルは以下の処理に利用されます。
- 銘柄コードから紐づくテーマタグ群の取得（`code_to_tags` 辞書の構築）
- 特定のテーマタグから、そこに所属するすべての銘柄コード群の逆引き（`tag_to_codes` 辞書の構築）

---

## 2. データ構造（スキーマ定義）
JSON は配列（Array）構造になっており、各銘柄オブジェクトは以下のフィールドを持ちます。

| フィールド名 | データ型 | 説明 |
| :--- | :--- | :--- |
| `code` | `string` | 4桁の銘柄コード（例: `"1301"`, グロース新規上場などアルファベット混在の場合は `"130A"` など） |
| `name` | `string` | 企業の正式名称（例: `"極洋"`） |
| `hira` | `string` | 企業名の平仮名読み（例: `"きょくよう"`） |
| `kana` | `string` | 企業名の片仮名読み（例: `"キョクヨウ"`） |
| `search_meta` | `string` | 検索用のスペース区切りメタ情報。ひらがな・カタカナ・漢字を連結したもの（例: `"きょくよう キョクヨウ 極洋"`） |
| `short_name` | `string` | 画面表示等で使用する簡略化された企業名（略称） |
| `unprocessed` | `boolean` | タグ生成などの前処理が未完了かどうかを示すフラグ（通常は `false`） |
| `tags` | `array of string` | 当該銘柄に付与されているすべてのテーマタグ（フラットな配列） |
| `layered_tags` | `object` | カテゴリ（レイヤー）ごとにタグを体系的に分類したオブジェクト（詳細は後述） |
| `active_layers` | `array of string` | `layered_tags` 内で、空ではない（タグが存在する）レイヤー名のリスト |
| `industry` | `string` | より大雑把な業界区分（例: `"加工食品"`, `"医薬品"` など） |

### 2.1. 階層化タグ（`layered_tags`）のカテゴリ定義
タグは性質に応じて、以下のカテゴリに分類されて格納されています。これらを活用することで、「大分類セクター」や「コア技術」に絞ったスコアリングなど、柔軟な分析が可能です。

*   **`Sector_Major`** (大分類業界・セクター)
    *   例: `"水産"`, `"農林水産"`, `"小売り"` など一般的な主要産業。
*   **`End_Market`** (需要先・エンドマーケット)
    *   例: `"冷凍食品"`, `"医療・医薬品"`, `"車載向け"` など、製品やサービスの最終供給先市場。
*   **`Core_Product_Tech`** (コア製品・固有技術)
    *   例: `"養殖"`, `"核酸医薬"`, `"セラミックコンデンサー"`, `"パワー半導体"` など、企業の競争力の源泉となる製品・技術。
*   **`Business_Model`** (ビジネスモデル)
    *   例: `"SaaS"`, `"サブスクリプション"`, `"OEM"` など。
*   **`Policy_Event`** (政策・国策・イベント)
    *   例: `"防衛力強化"`, `"DX推進"`, `"インバウンド"` など。
*   **`Geopolitics`** (地政学・マクロ経済)
    *   例: `"円安メリット"`, `"ロシア関連"`, `"中国経済影響"` など。
*   **`Capital_Corporate`** (資本移動・コーポレートイベント)
    *   例: `"2024年IPO"`, `"MBO"`, `"親子上場"` など。
*   **`Unknown`**
    *   分類が特定できないその他のタグ。

## 4. カテゴリ（レイヤー）別主要タグ一覧

辞書ファイルに定義されている各カテゴリの代表的なタグ、および `Sector_Major`（大分類業界・セクター）の全タグリストです。

### 4.1. `Sector_Major` (全93タグ)
`Sector_Major` に属するすべてのタグは以下の通りです（五十音順）。これらは、主に企業の基本的な産業区分（セクター）を決定する際の大枠として使用されます。

> BS放送, CS放送, IT関連, その他製造業, インスタント食品, インフラ, エネルギー, オーガニック食品, サービス業, スマート農業, ソフト・システム開発, デジタル印刷, ネット証券, ネット銀行, メガバンク, 不動産関連, 人体通信, 保険, 倉庫, 出版, 化学, 半導体商社, 印刷, 卸売, 商社, 土木, 地上デジタル放送, 地銀, 専門商社, 専門店, 小売り, 広告, 広告代理店, 建設, 建設・土木, 投資銀行, 損害保険, 放送, 放送局, 新聞, 日用雑貨卸, 有線放送, 林業, 染色加工, 機械, 水産, 水産卸, 海洋土木, 海運, 消費者金融, 物流, 特殊土木, 生命保険, 石油, 石油元売り, 石油化学, 米穀卸, 精密機器, 紙・パルプ, 紙卸, 素材, 紡績, 総合商社, 繊維, 衛星放送, 製粉, 製糖, 製紙, 製缶, 設備工事, 証券, 証券金融, 貿易, 農林水産, 通信, 通信キャリア, 造船, 運輸, 都市鉱山, 量子暗号通信, 金融, 鉄鋼, 鉄鋼商社, 銀行, 銀行事業・業務, 陸運, 電力会社, 電力線通信, 電気機器, 電炉, 非鉄, 食品, 高炉

### 4.2. その他のカテゴリの代表的なタグ

#### `End_Market` (需要先・エンドマーケット / 代表例)
製品の最終用途やサービス提供先市場を表すタグです。
- **デジタル・情報**: `EV関連`, `eスポーツ`, `オンラインゲーム`, `ゲーム関連`, `コンテンツ配信` など
- **小売・サービス**: `100円ショップ`, `eコマース`, `インバウンド`, `コンビニエンスストア`, `旅行` など
- **自動車・インフラ**: `その他自動車関連`, `車載向け`, `オフィス回帰` など

#### `Core_Product_Tech` (製品・技術 / 代表例)
企業固有の競争力・独自技術を表すタグで、全カテゴリの中で最も細分化されています。
- **エレクトロニクス・半導体**: `セラミックコンデンサー`, `パワー半導体`, `EUV`, `CMOSセンサー`, `HBM`, `5G`, `6G`, `LED製造装置` など
- **IT・ソフトウェア**: `AIエージェント`, `AI医療診断`, `CRM`, `ERP`, `IoT`, `HRテック` など
- **その他**: `3Dプリンター`, `CAD`, `DNAチップ` など

#### `Business_Model` (ビジネスモデル / 代表例)
収益構造や業務形態を表すタグです。
- `SaaS`, `サブスク`, `OEM`, `SPA`, `CDMO`, `フランチャイズ展開`, `ファブレス`, `アウトソーシング`, `人材派遣`, `BtoB` など

#### `Policy_Event` (政策・国策・イベント / 代表例)
法改正や社会的なトレンド・イベントを表すタグです。
- `DX`, `2024年問題`, `カーボンニュートラル`, `マイナンバー関連`, `働き方改革`, `インボイス制度`, `ふるさと納税`, `リスキリング` など

#### `Geopolitics` (地政学・マクロ経済 / 代表例)
為替、国際関係、特定地域に関連するタグです。
- `円安メリット`, `円高メリット`, `原油高メリット`, `中国関連`, `インド関連`, `ロシア関連`, `グローバルサウス`, `インフラ輸出` など

#### `Capital_Corporate` (コーポレートイベント / 代表例)
資本市場に関連するイベント・属性タグです。
- `2024年IPO`, `2025年IPO`, `M&A`, `株主優待`, `親子上場`, `ベンチャーキャピタル`, `優待人気` など
```
