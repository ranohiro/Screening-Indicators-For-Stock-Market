# 特定銘柄の事後検証用ノートブック（`forensic-analysis.ipynb`）仕様・解説書

このドキュメントは、日本株投資アプリの「注目テーマ自動検知ロジック」において、**「特定の過去期間に、特定の銘柄群を検知するためには、どのようなパラメータ設定（閾値）やデータに注目しておくべきだったのか？」**をボトムアップで逆算・検証するために作成されたノートブック [forensic-analysis.ipynb](../notebooks/forensic-analysis.ipynb) の技術仕様および使用方法をまとめたものです。

---

## 1. 目的と位置づけ
既存の `sector-detec-param.ipynb` が「指定したパラメータでどのようなテーマ・銘柄が抽出されるか」を確認するトップダウン型のアプローチであるのに対し、本ノートブックは「**あらかじめ抽出したい正解（銘柄と期間）が決まっている状態で、それらを拾い上げるための条件を事後分析（フォレンジック分析）する**」ボトムアップ型のアプローチを取ります。

主な目的は以下の通りです。
- **目標銘柄群の共通テーマ（タグ）の特定**: ターゲットとなる銘柄群が共通して持っているテーマタグを洗い出します。
- **株価・出来高推移の可視化**: 指定期間におけるターゲット銘柄群の値動きと出来高モメンタムを可視化し、動意づいたタイミングを視覚的に特定します。
- **必要閾値（パラメータ）の逆算**: それらの銘柄を検知するために必要だった「出来高急増倍率（`VOL_RATIO`）」や「高値圏ブレイクアウト（`PRICE_HIGH`）」の閾値を算出し、適切なパラメータ設計の指針を得ます。

---

## 2. 前提データ構造
計算には既存の検証と同様、以下の2つのデータソースを使用します。

1. **テーマ・タグマスター (`ticker_dictionary.json`)**
   - 各銘柄に付与されたテーマタグ情報を読み込み、指定した複数銘柄の共通タグを抽出するために利用します。
2. **日足株価データベース (`stock_data.db` 内の `daily_prices` テーブル)**
   - 全上場銘柄の日足データ（日付、終値、出来高など）から、ターゲット銘柄の指定期間のデータを取得して計算・可視化に利用します。

---

## 3. 分析のフロー（ノートブックの構成）

本ノートブックは以下のステップで分析を実行します。

### 【Step 1: ターゲット設定とデータロード】
- **ターゲットの定義**: 分析したい「特定銘柄群（銘柄コードのリスト）」と「対象期間（開始日〜終了日）」を指定します。
  - *例: 太陽誘電(6976), TDK(6762), 村田製作所(6981) / 期間: 2026/04/01 〜 2026/05/25*
- **データの読み込み**: ターゲット銘柄の指定期間を含む株価データと、タグマスターを読み込みます。

### 【Step 2: 共通テーマタグの抽出】
- ターゲット銘柄の各タグリストを取得し、積集合（Intersection）を取ることで、**「それら全銘柄に共通して付与されているタグ」**を洗い出します。
  - これにより、「セラミックコンデンサー」「地上デジタル放送」「電気機器」など、どのテーマに着目すればその銘柄群を捉えられたかが分かります。

### 【Step 3: 指標計算と可視化】
- 各銘柄について、動意判定に必要な以下の指標を計算します。
  - `vol_5d_avg`: 5日移動平均出来高
  - `vol_25d_avg`: 25日移動平均出来高
  - `vol_ratio`: 出来高モメンタム（`vol_5d_avg / vol_25d_avg`）
  - `high_250d`: 過去250日高値（年初来高値相当）
  - `price_vs_high`: 高値圏位置（`close / high_250d`）
- 期間中の「株価と出来高倍率（`vol_ratio`）」の推移をグラフ化し、どの日に資金が大きく流入したか（モメンタムが発生したか）を視覚的に確認します。

### 【Step 4: 必要パラメータの逆算】
- 指定期間において、各銘柄の「出来高倍率（`vol_ratio`）の最大値」と「高値圏位置（`price_vs_high`）の最大値」を抽出します。
- **「これらの銘柄を期間内に少なくとも1回は検知するために、閾値（`VOL_RATIO`, `PRICE_HIGH`）を最低限どこまで下げる（または上げる）必要があったか」**をボトムアップで逆算・提示します。

---

## 4. 使用方法（パラメータの変更）
ノートブック内の「ターゲット設定」セルにて、以下の変数を書き換えて実行してください。

```python
# 検出したい特定銘柄群（銘柄コードのリスト）
TARGET_CODES = ['6976', '6762', '6981'] # 太陽誘電, TDK, 村田製作所

# 特定しておきたかった期間
START_DATE = '2026-04-01'
END_DATE = '2026-05-25'
```

結果として、共通タグ、各銘柄の推移グラフ、および「この銘柄群を検知するための推奨閾値」が出力されます。この結果を参考に、`sector-detec-param.ipynb` のパラメータを調整してください。
